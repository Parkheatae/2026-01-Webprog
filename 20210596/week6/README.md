# week06 학습 정리 4월 8일

## 1. 클래스와 객체

- `Rectangle` 클래스를 만들어 사각형 정보를 저장하는 방법을 학습했다.
- 객체를 생성하고, 생성된 객체의 정보를 출력하는 방법을 연습했다.

## 2. static 변수

- 객체가 생성될 때마다 전체 개수를 증가시키기 위해 `static` 변수를 사용했다.
- `static` 변수는 모든 객체가 공유하는 클래스 변수라는 점을 확인했다.

## 3. 생성자와 오버로딩

- 좌표와 크기를 직접 전달받는 생성자를 만들었다.
- 값이 주어지지 않을 때는 기본 생성자를 이용해 무작위 좌표와 크기를 생성했다.
- 생성자 오버로딩을 통해 다양한 방식으로 객체를 생성할 수 있음을 배웠다.

## 4. 난수 생성

- `Math.random()`과 `Random` 클래스를 사용하여 좌표와 크기를 무작위로 생성했다.
- 상수를 이용해 좌표와 크기의 범위를 제한했다.

## 5. 추상 클래스

- `Shape`를 추상 클래스로 만들고, 공통 속성과 메서드를 정의하는 방법을 학습했다.
- `Rectangle`, `Triangle`, `Circle`이 `Shape`를 상속하도록 구현했다.

## 6. 메서드 오버라이딩과 다형성

- 각 도형 클래스에서 `rotate()`, `show()` 등을 다르게 구현했다.
- 부모 타입인 `Shape`로 여러 자식 객체를 다루는 다형성을 연습했다.

## 7. 인터페이스

- 인터페이스를 이용해 `InterRectangle`, `InterTriangle`, `InterCircle` 클래스를 구현했다.
- 추상 클래스와 인터페이스의 차이를 비교해 보았다.

## 8. 익명 클래스

- `Shape`를 이용한 익명 클래스를 생성하여 필요한 메서드를 즉석에서 구현하는 방법을 배웠다.
- 이름이 없는 클래스를 한 번만 사용할 때 유용하다는 점을 확인했다.
