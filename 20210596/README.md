## 📝 오늘의 학습 (2026년 3월 26일)

- `week4` 폴더에 아래 파일 생성 및 구현
  1. `java_example.html`: 전역/지역 변수, 함수, `alert` 출력
  2. `if_while.html`: 1~10 반복, 짝/홀수 판별, `div#result`에 출력
  3. `function.html`: 함수 선언식/화살표 함수, 합계 계산, `div` 및 `alert` 출력
  4. `student.html`: `prompt` 입력(이름, 학번, 전공), 객체 + 메서드 자기소개
  5. `array_basic.html`: 10개 랜덤 점수 `for` 생성, 평균 계산, 60점 미만 `filter` 출력
  6. `change_image.html`: 버튼 클릭 시 이미지 교체(순환)
