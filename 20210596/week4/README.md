# Week4 학습 요약

- 날짜: 2026-03-26

## 구현 항목
1. `java_example.html`
   - 전역변수(`var`) 및 지역변수(`let`, `const`) 선언
   - 함수 내부에서 `alert`로 값 출력
2. `if_while.html`
   - `for` 반복문으로 1~10 숫자 순회
   - `if`로 짝수/홀수 판별
   - 결과를 `div#result`에 출력
3. `function.html`
   - 함수 선언식(`function`)으로 합계 계산
   - 화살표 함수(`=>`)로 합계 계산
   - 결과를 `div`에 표시 및 `alert`로 출력
4. `student.html`
   - `prompt`로 이름, 학번, 전공 입력 받음
   - 객체 생성 및 `introduce` 메서드로 자기소개 문장 반환
   - 결과를 `div#result`에 출력
5. `array_basic.html`
   - `for` 반복문으로 10개 랜덤 정수 생성 (0~100)
   - `for` 반복문으로 합계 계산 및 평균 산출
   - `filter`로 60점 미만 점수 추출
   - 결과를 `div#result`에 출력
6. `change_image.html`
   - 버튼 클릭 시 이미지 `src` 순환 교체

## 실행 방법
- `week4` 폴더의 각 HTML 파일을 브라우저로 열어서 동작 확인
- `student.html`은 입력창(`prompt`)이 나타나며, 결과가 화면에 표시됨
