## 2. 오늘 학습한 JSP 파일 요약
- `test.jsp`
  - JSP 기본 문법: `<%@ page ... %>`, `<%= ... %>`, `<% ... %>`
  - 현재 날짜/시간 출력 (Java `java.util.Date` 사용)
  - 요청 파라미터 `name` 읽기 및 기본값 처리
  - 사용자 인사 메시지 동적 생성
  - HTML 폼을 통해 이름 입력 후 재호출 기능 제공
  - EL 예시: `${pageContext.session.id}`
  - JSTL 예시 (`<c:choose>`, `<c:when>`, `<c:otherwise>`) -- JSTL 라이브러리 설치가 필요할 수 있음

## 3. 코드 스니펫 작성 요약
- JSP는 Java 서블릿과 HTML을 혼합하여 동적 페이지를 생성한다.
- 코드 스니펫을 작성할 때는 다음을 고려한다:
  1. 페이지 인코딩과 컨텐츠 타입 설정
  2. <% ... %>로 Java 로직 삽입
  3. <%= ... %>로 표현식 간단 출력
  4. EL (`${...}`)을 사용해 코드 가독성 향상
  5. JSTL과 커스텀 태그를 사용하면 로직을 JSP에서 분리하고 MVC 설계를 준수하는 데 도움이 된다
