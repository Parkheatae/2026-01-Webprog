## 2. 학습 파일별 정리

### 2-1. `Example.java`

기본적인 Java 문법을 종합적으로 연습한 파일이다.

#### 학습 내용
- 기본 변수 선언
- 배열 생성 및 초기화
- 문자열 연결과 포맷팅
- 일반 `for`문과 향상된 `for`문 사용
- 배열의 합계와 평균 계산

#### 주요 코드 설명
- `int`, `double`, `boolean` 자료형을 사용해 변수 선언을 연습했다.
- `scores`와 `subjects` 배열을 통해 여러 데이터를 묶어서 관리하는 방식을 익혔다.
- `"Hello, " + name + "!"` 형태의 문자열 연결과 `String.format()`을 사용한 문자열 포맷팅을 학습했다.
- 반복문으로 배열 요소를 순서대로 출력했다.
- 점수 총합을 구한 뒤 평균을 계산하는 방식도 확인했다.

#### 정리
이 파일을 통해 Java의 가장 기초적인 문법인 변수, 배열, 문자열, 반복문, 형변환, 계산 로직을 복습할 수 있었다.

### 2-2. `Multiples.java`

사용자로부터 정수를 입력받아, 그 수보다 작은 2부터 9까지의 배수를 출력하는 프로그램이다.

#### 학습 내용
- `Scanner`를 이용한 사용자 입력 처리
- 입력값 검증
- 조건문과 반복문 활용
- 중첩 반복문 사용
- 출력 형식 제어

#### 주요 코드 설명
- `Scanner` 객체를 사용해 사용자로부터 정수를 입력받았다.
- `hasNextInt()`를 통해 입력값이 정수인지 먼저 확인했다.
- 입력값이 2 이하일 경우 예외 메시지를 출력하도록 조건문을 작성했다.
- 바깥 반복문에서는 2부터 9까지의 수를 순회하고, 안쪽 반복문에서는 각 수의 배수를 차례대로 출력했다.
- 첫 번째 값인지 여부를 `boolean first`로 판단하여 쉼표 출력 형식을 조절했다.

#### 정리
이 파일을 통해 입력값 검증, 조건 분기, 반복문을 결합하여 원하는 규칙의 결과를 출력하는 방법을 익힐 수 있었다.

### 2-3. `PerfectNumbers.java`

2000 이하의 완전수를 찾고, 각 완전수의 약수를 함께 출력하는 프로그램이다.

#### 학습 내용
- 완전수 개념 이해
- 약수 탐색
- 누적합 계산
- 문자열 누적 처리
- 반복문을 통한 전체 탐색

#### 주요 코드 설명
- 2부터 2000까지의 수를 하나씩 검사하면서 완전수인지 판별했다.
- 각 숫자에 대해 자기 자신을 제외한 약수를 찾아 합계를 계산했다.
- 약수를 문자열로 함께 출력하기 위해 `StringBuilder`를 사용했다.
- 약수의 합이 원래 숫자와 같으면 완전수로 판정해 출력했다.

#### 정리
이 파일을 통해 약수의 합을 이용한 수학적 판별 로직을 구현했고, 반복문과 문자열 누적 처리 방식도 연습할 수 있었다.

### 2-4. `test.jsp`

JSP의 기본 문법과 동적 웹 페이지 작성을 실습한 파일이다.

#### 학습 내용
- JSP 페이지 지시어
- 선언문, 스크립틀릿, 표현식
- 현재 날짜 및 시간 출력
- 요청 파라미터 처리
- 사용자 입력을 반영한 동적 메시지 생성
- HTML form과 JSP 연동
- EL(Expression Language) 사용
- JSTL 태그 예시 확인

#### 주요 코드 설명
- `<%@ page ... %>`를 통해 페이지 설정 정보를 지정할 수 있음을 확인했다.
- `<%! ... %>`는 선언문, `<% ... %>`는 스크립틀릿, `<%= ... %>`는 표현식이라는 점을 구분해서 학습했다.
- `java.util.Date`를 사용해 현재 날짜와 시간을 출력할 수 있음을 확인했다.
- 요청 파라미터를 읽어 사용자 입력값에 따라 다른 내용을 출력하는 동작을 확인했다.
- HTML form을 사용해 값을 입력받고 다시 JSP에서 처리하는 흐름을 학습했다.
- `${pageContext.session.id}` 같은 EL 문법을 통해 값을 간단히 출력하는 방식을 확인했다.
- `<c:choose>`, `<c:when>`, `<c:otherwise>` 같은 JSTL 태그를 통해 조건 처리를 더 깔끔하게 할 수 있음을 이해했다.

#### 정리
이 파일을 통해 JSP가 HTML 안에 Java 코드를 넣어 동적인 페이지를 만들 수 있다는 점을 확인했다. 또한 코드가 복잡해질수록 스크립틀릿보다는 EL과 JSTL을 활용하는 것이 가독성과 유지보수 면에서 더 좋다는 점도 함께 배웠다.
