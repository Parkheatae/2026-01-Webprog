CSS3 Transition과 Animation 예제

Transition 예제들

Transition은 요소의 상태 변화 시 부드러운 애니메이션 효과를 줍니다.

버튼 Transition: hover 시 배경색과 크기가 0.3초 동안 ease 타이밍으로 변화
박스 Transition: hover 시 테두리 색상과 그림자가 0.4초 동안 ease 타이밍으로 변화
글자 Transition: hover 시 색상과 글자 크기가 0.5초 동안 ease 타이밍으로 변화

Animation 예제들

Animation은 @keyframes를 사용하여 복잡한 애니메이션 시퀀스를 정의합니다.

박스 회전 Animation: rotate-box 키프레임으로 2초 동안 무한 회전 (linear 타이밍)
글자 색상 변화 Animation: color-change 키프레임으로 3초 동안 색상 순환 (ease-in-out 타이밍)
색상이 변하는 글자
버튼 Pulsate Animation: pulsate 키프레임으로 1.5초 동안 크기와 투명도 변화 (ease-in-out 타이밍)
박스 이동 Animation: slide 키프레임으로 2초 동안 좌우 이동 (ease-in-out 타이밍)

요약

  • Transition: 상태 변화(주로 :hover) 시 자동으로 적용되는 간단한 애니메이션
  • Animation: @keyframes로 세밀하게 제어되는 복잡한 애니메이션 시퀀스
  • 타이밍 함수: ease, linear, ease-in-out 등으로 애니메이션 속도 곡선 조절
  • 반복: infinite로 무한 반복, 또는 특정 횟수 지정 가능
