## 📝 오늘의 학습 (2026년 3월 19일)

- **부트스트랩 반응형 웹 페이지 예제 생성**: `bootstrap.html` 파일에 Bootstrap 5.3.3을 사용하여 반응형 웹 페이지를 구성했습니다.
  - 사용된 컴포넌트: 컨테이너, 그리드 시스템 (container, row, col-*), 네비게이션 바, 카드, 폼, 버튼
  - 반응형 디자인 적용: 브라우저 창 크기 조절 시 레이아웃 변경 확인 가능
  - 각 컴포넌트의 클래스 역할에 대한 HTML 주석 추가

- **박스 모델 실습**: `box_model.html` 파일에서 CSS 박스 모델 (margin, border, padding, content)의 개념과 적용 방법을 실습했습니다.

- **CSS 기본 실습**: `css_basic.html` 파일에서 CSS 선택자, 속성, 값 등을 사용하여 기본적인 스타일링을 연습했습니다.

- **플렉스 디스플레이 실습**: `flex_display.html` 파일에서 CSS Flexbox를 활용하여 유연한 레이아웃을 구성하고 실습했습니다.

- **폼, 리스트, 테이블 실습**: `form_list_table.html`과 `form_list_table.css` 파일에서 HTML 폼 요소, 리스트, 테이블을 생성하고 CSS로 스타일링하는 방법을 실습했습니다.
